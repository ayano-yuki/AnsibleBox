# test-update
Ansibleのplaybookを書く練習で作った「aptをアップデート」playbookです。

## 実行コマンド
```sh
ansible-playbook -i inventory.yaml playbook.yaml
```

## 出力結果
```sh
$ ansible-playbook -i inventory.yaml playbook.yaml

PLAY [aptのアップデート] ****************************************************************************************************************************************

TASK [Gathering Facts] ******************************************************************************************************************************************
ok: [conoha]

TASK [Update apt package list] **********************************************************************************************************************************
changed: [conoha]

TASK [Upgrade all packages] *************************************************************************************************************************************
ok: [conoha]

PLAY RECAP ******************************************************************************************************************************************************
conoha                     : ok=3    changed=1    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0 
```
