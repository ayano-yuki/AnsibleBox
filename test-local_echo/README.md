# test-local_echo
AnsibleのPlaybook実行中に、制御ノードでechoの結果を表示させることは可能かを確かめた。
（結果、可能）

## 実行コマンド
```sh
ansible-playbook -i inventory.yaml playbook.yaml
```

## 出力結果
```sh
$ ansible-playbook -i inventory.yaml playbook.yaml

PLAY [制御ノードに実行結果を表示させたい] ******************************************************************************************************************

TASK [Gathering Facts] *************************************************************************************************************************************
ok: [conoha]

TASK [Echo a message] **************************************************************************************************************************************
changed: [conoha]

TASK [Show echo result] ************************************************************************************************************************************
ok: [conoha] => {
    "echo_result.stdout": "Hello from control node!"
}

PLAY RECAP *************************************************************************************************************************************************
conoha                     : ok=3    changed=1    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0
```
